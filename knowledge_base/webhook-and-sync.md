# Webhook 与数据同步问题

## Webhook 没有收到回调

Webhook 没有收到回调时，请检查：

1. 回调地址必须是公网可访问的 HTTPS 地址。
2. 业务系统需要在 5 秒内返回 2xx 状态码。
3. 如果连续失败，平台会暂停该 Webhook，需要在控制台重新启用。
4. 请确认事件类型已经在控制台订阅。

平台会对失败的 Webhook 进行最多 3 次重试，每次重试间隔会逐渐增加。

## 数据同步延迟

数据同步存在 1 到 5 分钟延迟属于正常情况。延迟超过 10 分钟时，建议：

1. 检查同步任务是否处于启用状态。
2. 查看最近一次同步日志中的错误信息。
3. 确认源系统账号没有被禁用。
4. 如果同步数据量很大，请分批同步或调整同步时间窗口。
