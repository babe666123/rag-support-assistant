# 套餐、部署与环境变量

## 套餐额度用完

当套餐额度用完后，接口可能返回额度不足错误。处理方式：

1. 在控制台查看本月已用额度和剩余额度。
2. 升级套餐或购买额外资源包。
3. 对非核心任务做缓存，减少重复请求。
4. 对批处理任务设置每日调用上限。

## 本地部署后无法调用模型

本地部署后无法调用模型时，优先检查环境变量：

1. 项目根目录是否存在 `.env` 文件。
2. `.env` 中是否配置 `DEEPSEEK_API_KEY`。
3. API Key 是否写在后端环境中，而不是前端 JavaScript 文件中。
4. 是否能访问 `https://api.deepseek.com/chat/completions`。

API Key 不应该提交到 GitHub。请将 `.env` 加入 `.gitignore`。
